一、node js的安装

进入Node.js官网，网址https://nodejs.org/en/，点击download，按照下图箭头指示操作，下载好安装包。


下载好的安装包，打开它。

安装的时候，建议不要安装在系统盘，可以安装在D盘或者其他盘

点击 Next 进行下一步

再打勾，选Next进行下一步



点击 Finish ，安装完成。

下面进行测试安装是否成功，输入cmd 打开命令提示符，然后输入 node -v，若出现对应的版本说明，则说明安装成功了。


成功安装后的目录如下图

注意：新版的Node.js已自带npm，安装Node.js时会一起安装，npm的作用就是对Node.js依赖的包进行管理，也可以理解为用来安装/卸载Node.js需要装的东西

路径配置

说明：这里的环境配置主要配置的是npm安装的全局模块所在的路径，以及缓存cache的路径，之所以要配置，是因为以后在执行类似：npm install express [-g] （后面的可选参数-g，g代表global全局安装的意思）的安装语句时，会将安装的模块安装到【C:\Users\用户名\AppData\Roaming\npm】路径中，占C盘（也就是系统盘）空间。

我是将全模块所在路径和缓存路径放在我node.js安装的文件夹中，则在我安装的文件夹【D:\ProgramFile\nodejs】下创建两个文件夹【node_global】及【node_cache】如下图：

接下来，打开cmd命令窗口，以我自己的安装路径为例，输入

npm configsetprefix”D:\Program File\nodejs\node_global”

npm configsetcache”D:\Program File\nodejs\node_cache”

注意：后面的路径是你node.js的安装路径。


接下来，进行环境变量的配置

关闭命令窗口。按下图箭头所示，打开环境变量

在【系统变量】下新建【NODE_PATH】，输入【D:\Program File\nodejs\node_global\node_modules】

【用户变量】下的【Path】修改为【D:\Program File\nodejs\node_global】
